f(x) = \begin{cases}
x & \text{if } x \geq 0 \\
A(e^x - 1) & \text{if } x < 0
\end{cases}
\] and its derivative is\+:  \[
f\textnormal{\textquotesingle}(x) = \begin{cases}
1 & \text{if } x \geq 0 \\
f(x) + A & \text{if } x < 0
\end{cases}
\] where $A$ is a constant.
