template<typename T , typename W , template< typename F > typename Activator, template< typename E, typename LW, template< typename K > typename Act > typename Loss>
void Network< T, W, Activator, Loss >::forward_propatation ( )

Forward propagate the values of the network.

This function is used to forward propagate the values of the network. The values of the input layer are used as input to the network. The values of the output layer are stored in the output layer of the network.
