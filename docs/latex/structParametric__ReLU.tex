The Re\+LU activation function. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & The type of the input.\\
\hline
\end{DoxyTemplParams}
The Re\+LU activation function is defined as\+:  \[
 f(x) = \max(0, x)
 \] and its derivative is\+:  \[
 f\textnormal{\textquotesingle}(x) = \begin{cases}
 1 & \text{if } x \geq 0 \\
 0 & \text{if } x < 0
 \end{cases}
 \]
